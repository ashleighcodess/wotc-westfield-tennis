# Tennis Together | Westfield Indoor Tennis Club
## Project Brief

---

### Client
**Marcy** - Building a women's tennis community at Westfield Indoor Tennis Club

---

### Core Vision
One central place where a growing community of women can stay connected, informed, and engaged - without complicated systems or a learning curve.

---

### Client Pain Points (What to Avoid)
- WordPress complexity
- Plugin management
- CRM confusion
- Software updates
- Heavy back-end tools
- Steep learning curves

---

### What Marcy Needs to Do (Dashboard Requirements)

| Task | Dashboard Feature |
|------|------------------|
| Post blogs and updates | Simple blog editor |
| Share videos | Video embed (YouTube/Vimeo links) |
| Share announcements | Quick announcement posts |
| Add/update events | Event manager with date, description, registration link |
| Add/update programs | Program manager with details, pricing, Stripe link |
| Control what's live | Toggle visibility on/off |
| Link payments | Stripe payment link integration |
| Sell products/equipment | Product manager with Stripe checkout |
| Affiliate content | Blog posts with affiliate links |

---

### Public Website Pages

1. **Home** - Welcome, upcoming events preview, community highlights
2. **About** - Story, mission, team/leadership
3. **Community & Events** - Calendar, upcoming events, announcements, community updates
4. **Programs** - Tennis programs, clinics, lessons with registration/payment
5. **Shop** - Equipment, apparel, affiliate products
6. **Contact** - Contact form, location, social links

---

### Monetization Streams

1. **Events** - Paid event registration via Stripe
2. **Programs** - Program enrollment payments via Stripe
3. **Products** - Equipment/merchandise sales via Stripe
4. **Affiliates** - Blog content with affiliate product links

---

### Technical Approach

**Why Custom-Built (No CMS):**
- No plugins to update or break
- No WordPress security patches
- No CRM subscriptions
- Purpose-built = only what's needed
- Simple interface = no learning curve
- Fast, lightweight, future-proof

**Stack:**
- Static HTML/CSS/JS (public site)
- Netlify hosting + serverless functions
- JSON data storage (Netlify Blobs or JSON files)
- Stripe for all payments
- Simple token-based auth for dashboard

---

### Dashboard Design Principles

1. **One-click actions** - Toggle visibility, publish, unpublish
2. **No jargon** - "Add Event" not "Create New Event Instance"
3. **Visual feedback** - Green = live, Gray = hidden
4. **Mobile-friendly** - Marcy can update from her phone
5. **Minimal fields** - Only ask for what's necessary
6. **Pre-built templates** - Event form, blog post form, product form

---

### Social Media Integration

- Instagram feed embed
- Facebook integration
- Social sharing buttons
- Content structured for easy social posting

---

### Success Metrics

- Marcy can post an update in under 2 minutes
- Marcy can add an event without asking for help
- Community members can find events and register easily
- Payments flow through Stripe seamlessly
- Site loads fast on mobile
